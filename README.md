##  Overview

**AURORA** is an interactive 3D particle system that responds to real-time hand gestures detected through your webcam.



---

##  Gesture Controls

| Fingers Extended | Particle Pattern |
|------------------|------------------|
| 0 | Saturn |
| 1 | Cat |
| 2 | "HEY" |
| 3 | face |
| 4 | Heart |
| 5 | Random Nebula |

Gesture detection is stabilized using a short finger history buffer to prevent flickering.
